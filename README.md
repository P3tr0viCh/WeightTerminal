# Весовой терминал
Поддержка:
- Ньютон-42
  - Посылка сообщения (Prn1)
  - Сжатая двоичная посылка (bIn)
- МИДЛ МИ ВДА/12Я
- Микросим М0601
- Токвес SH-50
- HBM WE2110: HBM стандартный формат
  - По умолчанию. Настройки автопередачи AUT.OPT: Type = Auto.A, St.Chr = 02, End.ch1 = 03, End.ch2 = 00
  - Альтернативная конфигурация. Настройки автопередачи AUT.OPT: Type = Auto.A, St.Chr = 00, End.ch1 = 13, End.ch2 = 10
